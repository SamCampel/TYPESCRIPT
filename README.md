# Introdução ao TypeScript

Este README serve como um guia de referência para aula introdutória sobre TypeScript que assisti no YouTube. O objetivo foi me fornecer uma visão geral e conceitos básicos do TypeScript, uma linguagem de programação que é um superconjunto de JavaScript, adicionando tipagem estática opcional e outras funcionalidades que ajudam na escrita de código mais robusto e manutenível.
Onde pude ter as noções básicas de desenvolvimentos com esssa ferramenta.
